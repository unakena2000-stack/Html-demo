# HTML Demo Project

A comprehensive demonstration of modern web development using HTML5, CSS3, and vanilla JavaScript.

## 📁 Project Structure

```
Html-demo/
├── index.html       # Main landing page
├── about.html       # About page with project details
├── contact.html     # Contact form page
├── styles.css       # Global stylesheet
├── script.js        # JavaScript functionality
└── README.md        # This file
```

## 🚀 Features

- **Responsive Design**: Mobile-first approach with responsive grid layouts
- **Semantic HTML5**: Proper HTML structure using semantic elements
- **Modern CSS3**: Flexbox, Grid, animations, and gradients
- **Vanilla JavaScript**: No frameworks, pure JavaScript interactivity
- **Accessibility**: WCAG compliant design principles
- **Navigation**: Multi-page navigation with active link highlighting
- **Forms**: Contact form with basic validation
- **Animations**: Smooth transitions and animations throughout

## 📄 Pages

### Home (index.html)
- Welcome banner with call-to-action button
- Features showcase grid
- Responsive hero section with gradient background

### About (about.html)
- Project mission and overview
- Key concepts covered (Semantic HTML, CSS, JavaScript, etc.)
- Feature highlights with icons

### Contact (contact.html)
- Contact form with validation
- Contact information display
- Form submission handling with confirmation

## 🎨 Styling

The project uses a modern color scheme:
- Primary Color: `#3498db` (Blue)
- Dark Color: `#2c3e50` (Dark Slate)
- Gradients and hover effects for interactivity

## 💻 Technical Details

### HTML
- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<footer>`, etc.)
- Meta tags for viewport and charset
- Proper form structure with labels

### CSS
- CSS Grid for responsive layouts
- Flexbox for component alignment
- CSS animations and transitions
- Media queries for mobile responsiveness
- CSS variables for maintainability

### JavaScript
- DOM manipulation
- Event listeners
- Form handling
- Dynamic link highlighting
- Console logging for debugging

## 🔧 How to Use

1. Clone the repository
2. Open `index.html` in a web browser
3. Navigate through the pages using the navigation menu
4. Test the contact form on the Contact page
5. Try different screen sizes to see responsive design in action

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: Below 768px
